## 🚀 Getting Started

### 1. **Clone the Repository**
```bash
git clone https://github.com/bandarusiva955/protfolio.git
cd protfolio
```

### 2. **Open Locally**
Simply open `portfolio.html` in your browser:
```bash
open portfolio.html
```

Or use a simple HTTP server:
```bash
python -m http.server 8000
# Visit http://localhost:8000
```

### 3. **Deploy to GitHub Pages**
The portfolio is already set up for GitHub Pages! Just:

1. Push the code to your `protfolio` repository
2. Go to **Settings** → **Pages**
3. Set source to `main` branch and `/ (root)` directory
4. Your portfolio goes live at: `https://bandarusiva955.github.io/protfolio/`
